Project Information

This project is focused on creating an interactive UI for a robot designed to operate on a college campus. The robot will be able to interact with four distinct user types: Admin, Faculty, Students, and Visitors. Each user type will have access to different functionalities, with Admin, Faculty, and Student pages being password-protected for secure access.

The landing page features an attractive robot face with large eyes and four interactive options for each user type. The UI is designed to be engaging, simple to navigate, and responsive across different devices. This system aims to provide seamless interaction between the robot and campus users, enhancing the overall experience.
